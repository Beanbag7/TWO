## Technology Stack

### Frontend

- Vue 3 (Composition API)
- Naive-ui
- Tailwind CSS
- remixicon
- Vue Router (Routing)
- Pinia (State Management)
- Axios (HTTP Client)
- Vite (Build Tool)

### Backend

- Spring Boot 
- Raw JDBC (with custom JDBCUtils)
- Spring Validation
- Global Exception Handling
- MySQL 8.0+
